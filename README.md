# process_helper

Makes it easier to spawn ruby sub-processes with proper capturing of stdout and stderr streams.

## Installation

Add this line to your application's Gemfile:

    gem 'process_helper'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install process_helper
